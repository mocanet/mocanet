## 概要
* .NET 開発で使用できるフレームワーク
* 複数のフレームワークのサポート
* WindowsおよびWeb用のユーティリティメソッドおよびカスタムコントロール
* NuGetを使用して各プロジェクトに追加
* テンプレートおよびスニペット用のVisual Studio拡張機能
